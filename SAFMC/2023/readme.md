# SAFMC-2023

https://www.youtube.com/watch?v=Xq0oRhGTwyc

Welcome to the official repository for Tellix (our Team Name for the competition SAFMC) a specialized search and rescue drone swarm developed by our team where we represent Nanyang Polytechnic for the SAFMC commpetition. Our mission was to engineer an autonomous system using a swarm of 10-25 drones capable of navigating complex arenas to identify and locate Victims in disaster-simulated environments.
